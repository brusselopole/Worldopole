# Worldopole

[![Build Status](https://scrutinizer-ci.com/g/brusselopole/Worldopole/badges/build.png?b=master)](https://scrutinizer-ci.com/g/brusselopole/Worldopole/build-status/master) [![Scrutinizer Code Quality](https://scrutinizer-ci.com/g/brusselopole/Worldopole/badges/quality-score.png?b=master)](https://scrutinizer-ci.com/g/brusselopole/Worldopole/?branch=master) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Based on [Brusselopole](http://www.brusselopole.be), Worldopole is an open source version of the website, allowing you to display in a nice way the data you have gathered of the Pokemon in your city such as Team Gym Battles, Pokemon nests and Pok√©stops. This app is a base and will help you build your own version of Brusselopole with more data-visualisation! 

Up-to-date demos:
- [Berlinopole](https://obihoernchen.net/pokemon/)
- [Default examlpe](http://pokemon.32k.ch/)

Chat with us on our official Discord: https://discord.gg/SDvsxkN

## Requirements
- PHP5+
- Apache2
- Mod_Rewrite (htaccess)
- Existing [PokemonGo-Map](https://github.com/PokemonGoMap/PokemonGo-Map) **MySQL** database
